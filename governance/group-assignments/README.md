# Group Assignments

Assigns IAM Identity Center groups to QuickSight roles and applies custom permission profiles.

## Resources

- `aws_quicksight_role_membership` — ADMIN_PRO and READER_PRO role assignments
- `aws_quicksight_custom_permissions` — restricts sharing for READER_PRO
- `aws_quicksight_role_custom_permission` — attaches custom permissions to READER_PRO role
